# Minimum Sum of Distances on a Number Line

Given two fixed points **A(1)** and **B(3)** on a number line, and a moving point **P(x)** on the same number line, investigate:

What is the minimum value of  
**|x - 1| + |x - 3|**?

If the minimum exists, determine:

1. The minimum value.
2. The position(s) of point **P** for which the minimum is attained.
